= 验证 SSH 配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果要使用以太网交换机运行状况监控器(CSHM)和日志收集功能、请验证是否已在集群交换机上启用SSH和SSH密钥。

.步骤
. 验证是否已启用 SSH ：
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. 验证是否已启用SSH密钥：
+
`show ssh key`

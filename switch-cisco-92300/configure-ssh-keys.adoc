= 请检查您的 SSH 配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您正在使用以太网交换机健康监视器 (CSHM) 和日志收集功能，请确认集群交换机上已启用 SSH 和 SSH 密钥。

.步骤
. 确认 SSH 已启用：
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. 请确认 SSH 密钥已启用：
+
`show ssh key`

+

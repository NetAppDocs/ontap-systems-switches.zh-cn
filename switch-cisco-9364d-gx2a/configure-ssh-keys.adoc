= 验证 SSH 配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您正在使用以太网交换机健康监视器 (CSHM) 和日志收集功能，请验证交换机上是否启用了 SSH 和 SSH 密钥。

.步骤
. 验证是否已启用 SSH ：
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. 验证是否已启用SSH密钥：
+
`show ssh key`

= Cisco Nexus 3232C 交换机的安装和设置工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]

[role="quick-margin-para"]
查看特定的交换机和控制器文档以设置您的 3232C 交换机和ONTAP集群。

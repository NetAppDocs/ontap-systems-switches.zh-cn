:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要配置SN2100交换机、请参阅NVIDIA的文档。

.步骤
. 查看 link:configure-reqs-sn2100-cluster.html["配置要求"]。
. 按照中的说明进行操作 https://docs.nvidia.com/networking/display/sn2000pub/System+Bring-Up["NVIDIA系统启动。"^]。


.下一步是什么？

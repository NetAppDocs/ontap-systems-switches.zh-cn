= 升级参考配置文件 (RCF) 脚本
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
请按照以下步骤升级 RCF 脚本。

.开始之前
升级 RCF 脚本之前，请确保交换机上具备以下条件：

* Cumulus Linux 已安装。参见 https://hwu.netapp.com/Switch/Index["Hardware Universe"^]适用于支持的版本。
* IP 地址、子网掩码和默认网关通过 DHCP 定义或手动配置。



NOTE: 除了管理员用户之外，您还必须在 RCF 中指定一个专门用于日志收集的用户。

.当前 RCF 脚本版本
集群和存储应用可以使用两种 RCF 脚本。从 https://mysupport.netapp.com/site/info/nvidia-cluster-switch["NVIDIA SN2100 软件下载"^]页。每种情况的处理步骤都相同。

* 集群：*MSN2100-RCF-v1._x_-Cluster-HA-Breakout-LLDP*
* 存储：*MSN2100-RCF-v1._x_-存储*


.关于示例
以下示例步骤展示了如何下载和应用集群交换机的 RCF 脚本。

示例命令输出使用交换机管理 IP 地址 10.233.204.71，子网掩码 255.255.254.0 和默认网关 10.233.204.1。

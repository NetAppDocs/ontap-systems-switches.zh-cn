= NVIDIA SN2100交换机的组件和部件号
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
对于NVIDIA SN2100 交换机的安装和维护，请务必查看机柜和导轨套件的组件和部件号列表。



== 内阁细节

您可以使用交换机附带的标准支架将NVIDIA SN2100 交换机 (X190006) 安装到NVIDIA双/单交换机机柜中。

= 使用NetApp CN1610 集群交换机的双节点交换集群的迁移选项
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您有一个双节点无交换机集群，您可以无中断地迁移到包含NetApp CN1610 集群网络交换机的双节点交换集群。具体操作步骤取决于每个控制器上是有两个专用集群网络端口还是只有一个集群端口。

大多数系统需要在每个控制器上设置两个专用的集群网络端口。

FAS22xx 节点允许每个控制器上只有一个集群端口。

有两种迁移方案可供选择：

* link:migrate-switched-netapp-cn1610.html["从无交换机集群迁移到有交换机的NetApp CN1610 集群环境"]
* link:migrate-fas22xx-systems.html["从无交换机集群（具有单个集群网络连接的 FAS22xx 系统）迁移"]

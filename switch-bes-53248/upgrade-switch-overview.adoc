= BES-53248 集群交换机的升级工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
请按照以下步骤升级 Broadcom BES-54328 集群交换机上的 EFOS 软件和参考配置文件 (RCF)（如适用）。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一个"]link:upgrade-efos-software.html["升级您的 EFOS 版本"]
[role="quick-margin-para"]
在 BES-53248 集群交换机上下载并安装以太网结构操作系统 (EFOS) 软件。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["两个"]link:upgrade-rcf.html["升级您的 RCF 版本"]
[role="quick-margin-para"]
升级 BES-53248 集群交换机上的 RCF，然后在应用 RCF 后验证端口是否具有额外的许可证。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:replace-verify.html["升级后验证ONTAP集群网络"]
[role="quick-margin-para"]
升级 EFOS 软件或 RCF 后，验证 BES-53248 集群交换机的ONTAP集群网络的运行状况。

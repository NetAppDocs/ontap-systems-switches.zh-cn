= 使用 AFX 系统、集群、存储和共享交换机启动并运行
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要启动并运行 AFX 系统、集群、存储和共享交换机，您需要安装硬件组件并配置交换机。

部署交换机涉及以下工作流程。

[role="quick-margin-para"]

= 交换机运行状况监控器概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以太网交换机运行状况监控器(CSHM)负责确保集群和存储网络交换机的运行状况、并收集交换机日志以进行调试。

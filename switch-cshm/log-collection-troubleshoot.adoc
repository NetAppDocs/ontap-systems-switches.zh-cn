= 排查日志收集问题
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您遇到日志收集功能报告的以下任何错误状态（可在输出中查看）， `system switch ethernet log show`命令），尝试相应的调试步骤：

|===


| *日志收集错误状态* | *解决* 


 a| 
*RSA密钥不存在*
 a| 
重新生成ONTAP SSH 密钥。



 a| 
*切换密码错误*
 a| 
验证凭据，测试 SSH 连接，并重新生成ONTAP SSH 密钥。请查阅交换机文档或联系NetApp支持以获取说明。



 a| 
*FIPS 系统中不存在 ECDSA 密钥*
 a| 
如果启用了 FIPS 模式，则需要在交换机上生成 ECDSA 密钥，然后再重试。



 a| 
*发现已存在的日志*
 a| 
删除交换机上之前的日志收集文件。



 a| 
*交换机转储日志错误*
 a| 
请确保切换用户拥有日志收集权限。请参考以上先决条件。

|===

NOTE: 如果上述解决方案无效，请联系NetApp支持。
